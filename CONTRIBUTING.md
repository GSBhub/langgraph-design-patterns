# Contributing

![Badge](https://img.shields.io/badge/Contributions-Welcome-0ea5e9) ![Badge](https://img.shields.io/badge/Docs-Markdown--first-0f172a?logo=markdown&logoColor=fff)

## Quick rules
- Keep PRs small and focused (one idea per PR).
- Prefer Markdown docs with short sections and skimmable bullets.
- Add links to related patterns/examples when helpful.

## What we accept
- New patterns in `patterns/` (template-compliant, one main idea).
- New decision guides in `decision-guides/` (table-heavy, immediately actionable).
- Tiny examples in `examples/` (proof-of-concept, not a framework).
